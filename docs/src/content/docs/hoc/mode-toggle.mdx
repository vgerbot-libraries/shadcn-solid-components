This module’s **default export** is a button that calls `toggleColorMode` from `@kobalte/core`. It shows a sun icon in light mode and a moon icon in dark mode using `data-kb-theme` selectors.

## Usage

```tsx
import ModeToggle from "shadcn-solid-components/hoc/mode-toggle"

<ModeToggle />
```

There is no named export from this entry; use a default import.

`toggleColorMode` depends on Kobalte's color mode context, so the app root should use
`ColorModeProvider` from `@kobalte/core`, just like the docs demo app does. The icon visibility is
switched via `html[data-kb-theme=light|dark]` selectors, so make sure the root element receives the
theme attribute written by Kobalte.
