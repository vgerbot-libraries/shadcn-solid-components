A thin horizontal or vertical rule styled as an `hr` (or polymorphic equivalent) for layout separation.

## Usage

```tsx
import { Separator } from "shadcn-solid-components/components/separator"
```

```tsx
<Separator class="my-2" />
```

Pass `class` and orientation-related props as supported by the underlying Kobalte separator primitive.

## Exports

- `Separator`
- Types: `SeparatorProps`
