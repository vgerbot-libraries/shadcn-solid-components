`Kbd` styles a single key cap; `KbdGroup` groups multiple keys for chord hints.

## Installation

Install `shadcn-solid-components` and satisfy peer dependencies (`solid-js`, Tailwind CSS v4, `tw-animate-css`).

## Usage

```tsx
import { Kbd, KbdGroup } from "shadcn-solid-components/components/kbd"
```

## Exports

- `Kbd`, `KbdGroup`
- Types: `KbdProps`, `KbdGroupProps`
