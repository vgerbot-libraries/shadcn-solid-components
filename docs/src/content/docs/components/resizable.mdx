Resizable panels are powered by `@corvu/resizable` with styled `Resizable`, `ResizablePanel`, and `ResizableHandle` parts.

## Installation

```bash
pnpm add @corvu/resizable
```

## Usage

```tsx
import { Resizable, ResizableHandle, ResizablePanel } from "shadcn-solid-components/components/resizable"
```

Nest `ResizablePanel` regions inside `Resizable` and separate them with `ResizableHandle`. Sizing behavior follows the underlying Corvu API.

## Exports

- `Resizable`, `ResizablePanel`, `ResizableHandle`
- Types: `ResizableProps`, `ResizablePanelProps`, `ResizableHandleProps`

      </ResizablePanel>
      <ResizableHandle />
      <ResizablePanel initialSize={0.5} minSize={0.2}>
        <Resizable orientation="vertical">

              <span class="font-semibold">Two</span>
            </div>
          </ResizablePanel>

              <span class="font-semibold">Three</span>
            </div>
          </ResizablePanel>
        </Resizable>
      </ResizablePanel>
    </Resizable>
  )
}

export default ResizableDemo
