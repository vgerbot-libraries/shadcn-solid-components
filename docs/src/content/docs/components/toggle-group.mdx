`ToggleGroup` wraps Kobalte’s toggle group and passes `variant` / `size` to `ToggleGroupItem` via context, using the same `toggleButtonVariants` styles as the standalone `ToggleButton`.

## Usage

```tsx
import { ToggleGroup, ToggleGroupItem } from "shadcn-solid-components/components/toggle-group"
```

Set `variant="outline"` or `size` on `ToggleGroup`; items inherit unless overridden per item. Selection behavior follows Kobalte’s toggle-group API (`value`, `onChange`, `multiple`, etc.).

## Exports

- `ToggleGroup`, `ToggleGroupItem`
- Types: `ToggleGroupProps`, `ToggleGroupItemProps`
