`DescriptionList` renders a `dl` grid of label/value pairs. Items can span columns, use horizontal or vertical layout, and optionally show a copy control for string values.

## Installation

Install `shadcn-solid-components` and satisfy peer dependencies (`solid-js`, Tailwind CSS v4, `tw-animate-css`).

## Usage

```tsx
import { DescriptionList } from "shadcn-solid-components/hoc/description-list"

<DescriptionList
  title="Profile"
  columns={2}
  items={[
    { label: "Name", value: "Ada Lovelace" },
    { label: "Email", value: "ada@example.com", copyable: true },
  ]}
/>
```

`copyable` is most useful when `value` can be serialized to a string. For more complex values
rendered as JSX, use `render` for a custom presentation. `card` works well when you want the list
to match the rest of a page's card-based layout.

## API

### `DescriptionListProps`

Extends `div` props.

- `items` — `DescriptionItem[]`
- `columns` — `1` | `2` | `3` | `4` (default `2`)
- `bordered` — border around each cell group
- `layout` — `'vertical'` | `'horizontal'`
- `title`, `actions` — header row
- `card` — wrap in `Card` (default `false`)
- `locale` — partial copy-button strings

### `DescriptionItem`

- `label`, `value` (string, number, or JSX), `span`, `copyable`, `render`
