Collapsible uses Kobalte’s collapsible primitive for trigger and content with height animation classes.

## Installation

Install `shadcn-solid-components` and satisfy peer dependencies (`solid-js`, Tailwind CSS v4, `tw-animate-css`).

## Usage

```tsx
import { Collapsible, CollapsibleContent, CollapsibleTrigger } from "shadcn-solid-components/components/collapsible"
```

```tsx
<Collapsible>
  <CollapsibleTrigger>Toggle</CollapsibleTrigger>
  <CollapsibleContent>Hidden until expanded.</CollapsibleContent>
</Collapsible>
```

## Exports

- `Collapsible`, `CollapsibleTrigger`, `CollapsibleContent`
- Types: `CollapsibleProps`, `CollapsibleTriggerProps`, `CollapsibleContentProps`
