Skeleton renders a pulsing neutral block you can size with utilities until real content is ready.

## Usage

```tsx
import { Skeleton } from "shadcn-solid-components/components/skeleton"
```

Apply `class` (for example width/height or rounded corners) to match the eventual layout.

## Exports

- `Skeleton`
- Types: `SkeletonProps`
