The menubar stacks top-level menus with triggers, content regions, items, checkboxes, radio items, submenus, separators, and shortcuts.

## Installation

Install `shadcn-solid-components` and satisfy peer dependencies (`solid-js`, Tailwind CSS v4, `tw-animate-css`).

## Usage

```tsx
import {
  Menubar,
  MenubarContent,
  MenubarItem,
  MenubarMenu,
  MenubarPortal,
  MenubarTrigger,
} from "shadcn-solid-components/components/menubar"
```

## Exports

- `Menubar`, `MenubarPortal`, `MenubarMenu`, `MenubarGroup`, `MenubarRadioGroup`, `MenubarTrigger`, `MenubarContent`, `MenubarItem`, `MenubarCheckboxItem`, `MenubarRadioItem`, `MenubarGroupLabel`, `MenubarItemLabel`, `MenubarSeparator`, `MenubarShortcut`, `MenubarSub`, `MenubarSubTrigger`, `MenubarSubContent`
- Types: `MenubarProps` and other `*Props` types as defined in the source (the shortcut component shares its name with a props type alias)
