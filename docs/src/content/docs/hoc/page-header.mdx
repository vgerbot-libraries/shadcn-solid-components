`PageHeader` uses the breadcrumbs and separator components to produce a common admin-style header.

## Installation

Install `shadcn-solid-components` and satisfy peer dependencies (`solid-js`, Tailwind CSS v4, `tw-animate-css`).

## Usage

```tsx
import { Button } from "shadcn-solid-components/components/button"
import { PageHeader } from "shadcn-solid-components/hoc/page-header"

<PageHeader
  breadcrumbs={[{ label: "Home", href: "/" }, { label: "Settings" }]}
  title="Settings"
  description="Manage your account."
  actions={<Button>Save</Button>}
/>
```

### Title and actions without breadcrumbs

You can omit `breadcrumbs` entirely and keep only the title row plus right-aligned `actions`:

```tsx
import { Button } from "shadcn-solid-components/components/button"
import { PageHeader } from "shadcn-solid-components/hoc/page-header"

<PageHeader
  title="Members"
  description="Manage organization members and permissions."
  actions={
    <>
      <Button size="sm" variant="outline" type="button">
        Import
      </Button>
      <Button size="sm" type="button">
        Invite
      </Button>
    </>
  }
/>
```

## API

### `PageHeaderProps`

Extends `div` props.

- `breadcrumbs` — `BreadcrumbItem[]` (`label`, optional `href`)
- `title` — required
- `description` — string or JSX
- `actions` — JSX on the right
- `separator` — bottom `Separator` when `true` (default `false`)
