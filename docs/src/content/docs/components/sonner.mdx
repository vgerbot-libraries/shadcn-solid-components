The **Toaster** component wraps the `somoto` toast renderer, sets `theme` from Kobalte’s `useColorMode()`, and applies CSS variables so surfaces align with your popover and radius tokens. Default SVG icons are provided for success, info, warning, error, and loading toasts.

## Installation

`somoto` and `@kobalte/core` are pulled in as dependencies.

Mount **Toaster** once near the root of your app (for example beside your router). Trigger toasts with `toast` (and helpers such as `toast.success`) from `somoto`; that package is already a dependency of this library.

## Usage

```tsx
import { toast } from "somoto"
import { Toaster } from "shadcn-solid-components/components/sonner"

<>
  <Toaster />
  <App />
</>
```

```tsx
toast.success(<>Saved</>)
```

Props are forwarded to the underlying `somoto` `Toaster` (same parameter type as in the library). You may override `icons`, `style`, or other supported options; the wrapper supplies defaults for theme integration.
