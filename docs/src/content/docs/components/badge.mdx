Badges use `badgeVariants` for size and emphasis. The component is polymorphic so you can render a `span`, `a`, or another element via `as`.

## Usage

```tsx
import { Badge, badgeVariants } from "shadcn-solid-components/components/badge"
```

```tsx
<Badge variant="secondary">Beta</Badge>
```

## Exports

- `Badge`, `badgeVariants`
- Types: `BadgeProps`
