Breadcrumbs combine a Kobalte breadcrumbs root with list, items, links, separators, and an ellipsis helper for collapsed paths.

## Usage

```tsx
import {
  BreadcrumbList,
  Breadcrumbs,
  BreadcrumbsEllipsis,
  BreadcrumbsItem,
  BreadcrumbsLink,
  BreadcrumbsSeparator,
} from "shadcn-solid-components/components/breadcrumbs"
```

The root `Breadcrumbs` primitive supplies a default chevron separator; override with your own `separator` prop when needed.

## Exports

- `Breadcrumbs`, `BreadcrumbList`, `BreadcrumbsItem`, `BreadcrumbsLink`, `BreadcrumbsSeparator`, `BreadcrumbsEllipsis`
- Types: `BreadcrumbsProps`, `BreadcrumbListProps`, `BreadcrumbsItemProps`, `BreadcrumbsLinkProps`, `BreadcrumbsSeparatorProps`, `BreadcrumbsEllipsisProps`
