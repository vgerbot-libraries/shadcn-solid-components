`TagInput` reads tags from `value` (defaults to `[]` when omitted). Use `onChange` whenever the list should update—adding or removing tags only calls `onChange`. Enter commits the current input; Backspace on an empty input removes the last tag. When `suggestions` is set, the list filters as you type.

## Usage

```tsx
import { createSignal } from "solid-js"
import { TagInput } from "shadcn-solid-components/hoc/tag-input"

const [tags, setTags] = createSignal<string[]>([])

<TagInput
  value={tags()}
  onChange={setTags}
  suggestions={["react", "solid"]}
  max={5}
/>
```

### Custom validation

```tsx
import { createSignal } from "solid-js"
import { TagInput } from "shadcn-solid-components/hoc/tag-input"

const [tags, setTags] = createSignal<string[]>([])

<TagInput
  value={tags()}
  onChange={setTags}
  validate={t => t.length >= 2 && /^[a-z0-9-]+$/i.test(t)}
/>
```

### Locale preset

```tsx
import { createSignal } from "solid-js"
import { TagInput, zhCNLocale } from "shadcn-solid-components/hoc/tag-input"

const [tags, setTags] = createSignal<string[]>([])

<TagInput value={tags()} onChange={setTags} locale={zhCNLocale} />
```
