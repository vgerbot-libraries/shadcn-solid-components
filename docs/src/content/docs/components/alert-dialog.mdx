Alert dialogs present blocking confirmation UI: overlay, title, description, and primary/secondary actions wired to Kobalte’s alert dialog primitive.

## Installation

Install `shadcn-solid-components` and satisfy peer dependencies (`solid-js`, Tailwind CSS v4, `tw-animate-css`).

## Usage

```tsx
import {
  AlertDialog,
  AlertDialogAction,
  AlertDialogCancel,
  AlertDialogContent,
  AlertDialogDescription,
  AlertDialogFooter,
  AlertDialogHeader,
  AlertDialogOverlay,
  AlertDialogPortal,
  AlertDialogTitle,
  AlertDialogTrigger,
} from "shadcn-solid-components/components/alert-dialog"
```

Use `AlertDialogTrigger` to open, `AlertDialogContent` for the modal body, and `AlertDialogAction` / `AlertDialogCancel` as close buttons with appropriate button styles.

## Exports

- `AlertDialog`, `AlertDialogPortal`, `AlertDialogTrigger`, `AlertDialogOverlay`, `AlertDialogContent`, `AlertDialogHeader`, `AlertDialogFooter`, `AlertDialogTitle`, `AlertDialogDescription`, `AlertDialogAction`, `AlertDialogCancel`
