`EmptyState` shows an optional icon, title, description, and a single action slot.

## Installation

Install `shadcn-solid-components` and satisfy peer dependencies (`solid-js`, Tailwind CSS v4, `tw-animate-css`).

## Usage

```tsx
import { EmptyState } from "shadcn-solid-components/hoc/empty-state"

<EmptyState
  title="No results"
  description="Try adjusting your filters."
  action={<Button variant="outline">Reset</Button>}
/>
```

A common pattern is to let it fill the available area when a list or table has no data. `icon` can
be any icon component or illustration, and `action` typically holds a primary action such as
"Create" or "Reset filters".

  ```tsx
  return (
    <EmptyState
      icon={<span class="text-4xl">📂</span>}
  ```

```tsx
      action={

```

## API

### `EmptyStateProps`

Extends `div` props.

- `icon` — illustration or icon JSX
- `title` — required heading
- `description` — optional string or JSX
- `action` — optional CTA JSX
