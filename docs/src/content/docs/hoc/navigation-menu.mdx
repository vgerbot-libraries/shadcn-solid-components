`NavMenu` maps `menus` to `NavigationMenu` triggers and content. A group with only `href` (and no `items` / `content`) renders the trigger as a direct link.

## Usage

```tsx
import { NavMenu } from "shadcn-solid-components/hoc/navigation-menu"

<NavMenu
  menus={[
    {
      trigger: "Products",
      items: [
        { label: "Analytics", description: "Reports", href: "/analytics" },
      ],
    },
  ]}
/>
```

### Featured column and custom panel

```tsx
import { NavMenu } from "shadcn-solid-components/hoc/navigation-menu"

<NavMenu
  menus={[
    {
      trigger: "Solutions",
      featured: {
        label: "Platform",
        description: "One place for analytics and billing.",
        href: "/platform",
      },
      items: [
        { label: "Analytics", description: "Dashboards", href: "/analytics" },
        { label: "Billing", description: "Invoices", href: "/billing" },
      ],
    },
    {
      trigger: "Custom",
      content: () => <div class="p-4 text-sm">Any JSX as menu body.</div>,
    },
  ]}
  gutter={8}
/>
```

### Vertical menu

```tsx
import { NavMenu } from "shadcn-solid-components/hoc/navigation-menu"

<NavMenu orientation="vertical" class="max-w-xs" menus={[{ trigger: "Home", href: "/" }]} />
```

### Adjust trigger and content spacing

`gutter` controls the gap, in pixels, between the navigation trigger and the dropdown content. Use
`class` on the root to control overall width or alignment:

```tsx
import { NavMenu } from "shadcn-solid-components/hoc/navigation-menu"

<NavMenu
  class="justify-center"
  gutter={10}
  menus={[
    {
      trigger: "Product",
      items: [{ label: "Overview", href: "/product" }],
    },
  ]}
/>
```

## API

### `NavMenuProps`

- `menus` ŌĆö `NavMenuGroup[]`
- `orientation` ŌĆö `'horizontal'` | `'vertical'` (default `'horizontal'`)
- `class` ŌĆö root class
- `gutter` ŌĆö number, gap between trigger and viewport (default `6`)

### `NavMenuGroup`

- `trigger` ŌĆö button label
- `href` ŌĆö if set without `items` / `content`, trigger is a link
- `items` ŌĆö `NavMenuItem[]` grid content
- `content` ŌĆö custom content factory (overrides `items`)
- `contentClass`, `featured` ŌĆö layout helpers for the panel

### `NavMenuItem`

- `label`, optional `description`, `href`, `onClick`, `disabled`, `class`

## Exports

- `NavMenu` ŌĆö component
- `NavMenuProps`, `NavMenuGroup`, `NavMenuItem` ŌĆö types
