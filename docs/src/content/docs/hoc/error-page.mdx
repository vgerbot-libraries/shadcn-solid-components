`ErrorPage` renders a centered error state with a large status code (or custom icon), title, description, and action buttons. Five built-in variants provide sensible defaults for common HTTP errors so you can render a complete error page with a single prop.

## Usage

```tsx
import { ErrorPage } from "shadcn-solid-components/hoc/error-page"

<ErrorPage variant="404" />
```

When no `variant` is specified the component defaults to `'generic'`, which shows an exclamation
mark instead of a numeric status code. Every text ‚Äî title, description, and the CTA label ‚Äî can be
overridden via props.

### Custom title, description, and actions

Override the defaults for a fully branded error experience:

```tsx
import { Button } from "shadcn-solid-components/components/button"
import { ErrorPage } from "shadcn-solid-components/hoc/error-page"

<ErrorPage
  variant="500"
  title="Oops ‚Äî something broke!"
  description="Our team has been notified. Please try again in a moment."
  actions={
    <>
      <Button as="a" href="/">Home</Button>
      <Button variant="outline" onClick={() => location.reload()}>
        Retry
      </Button>
    </>
  }
/>
```

## API

### `ErrorPageProps`

Extends `div` props.

- `variant` ‚Äî `'404'` | `'403'` | `'500'` | `'503'` | `'generic'` (default `'generic'`)
- `title` ‚Äî custom heading (overrides variant default)
- `description` ‚Äî custom body text (overrides variant default)
- `icon` ‚Äî custom icon or illustration replacing the status code display
- `actions` ‚Äî custom action buttons (default: a "Go back home" link to `/`)

### Variant defaults

| Variant   | Status code | Title                  | Description                                                            |
| --------- | ----------- | ---------------------- | ---------------------------------------------------------------------- |
| `404`     | 404         | Page not found         | Sorry, the page you are looking for doesn't exist or has been moved.   |
| `403`     | 403         | Access denied          | You don't have permission to access this page.                         |
| `500`     | 500         | Internal server error  | An unexpected error occurred on the server.                            |
| `503`     | 503         | Service unavailable    | The service is temporarily unavailable.                                |
| `generic` | !           | Something went wrong   | An unexpected error occurred. Please try again later.                  |

## i18n
