`ModeToggleGroup` renders two `Button`s in a muted pill container. The active mode uses the default button variant; the inactive side uses ghost.

## Usage

```tsx
import { ModeToggleGroup } from "shadcn-solid-components/hoc/mode-toggle-group"

<ModeToggleGroup lightLabel="Light" darkLabel="Dark" />
```

### Wrapper layout

`ModeToggleGroup` forwards extra props to the root `div` (except `class`, which is merged with defaults).

```tsx
import { ModeToggleGroup } from "shadcn-solid-components/hoc/mode-toggle-group"

<ModeToggleGroup class="ml-auto" data-testid="theme-toggle" />
```

### Root forwards test and accessibility props

Aside from `class`, which is merged with the default styles, other `div` props such as
`data-testid` and `aria-label` are forwarded directly to the outer container:

```tsx
import { ModeToggleGroup } from "shadcn-solid-components/hoc/mode-toggle-group"

<ModeToggleGroup aria-label="Color mode" data-testid="mode-toggle-group" />
```

## API

### `ModeToggleGroupProps`

Extends `div` props.

- `lightLabel` — default `'Light'`
- `darkLabel` — default `'Dark'`
