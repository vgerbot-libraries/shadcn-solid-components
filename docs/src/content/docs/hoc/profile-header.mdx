`ProfileHeader` renders a card-based user profile with optional cover/banner image, avatar (image
or initials fallback), name, subtitle, bio, info items, stats row, action buttons, and tabbed
content panels powered by the `Tabs` component.

## Usage

```tsx
import { ProfileHeader } from "shadcn-solid-components/hoc/profile-header"

<ProfileHeader
  avatar="https://example.com/avatar.png"
  name="Jane Doe"
  subtitle="Product Designer"
  bio="Building beautiful interfaces."
  stats={[
    { label: "Followers", value: "1.2k" },
    { label: "Projects", value: 42 },
  ]}
/>
```

The avatar falls back to initials (from the `initials` prop, or auto-derived from the first two
characters of `name`). When `coverImage` is provided, a banner section with gradient overlay
renders above the avatar.

### Initials fallback (no avatar)

When no `avatar` URL is provided, a coloured circle with initials is rendered:

```tsx

  name="Sam Wilson"
  initials="SW"
  subtitle="Engineering Manager"
/>
```

### Controlled tabs

```tsx
import { createSignal } from "solid-js"
import { ProfileHeader } from "shadcn-solid-components/hoc/profile-header"

export function ControlledProfile() {
  const [tab, setTab] = createSignal("posts")

  return (
    <ProfileHeader
      name="Jane Doe"
      activeTab={tab()}
      onTabChange={setTab}
      tabs={[
        { id: "posts", label: "Posts", body: <PostsList /> },
        { id: "about", label: "About", body: <AboutSection /> },
        { id: "activity", label: "Activity", body: <ActivityLog /> },
      ]}
    />
  )
}
```

## API

### `ProfileHeaderProps`

Extends `div` props.

- `avatar` ‚Äî avatar image URL
- `initials` ‚Äî fallback initials (auto-derived from `name` if omitted)
- `name` ‚Äî **required** ‚Äî user display name
- `subtitle` ‚Äî secondary text (e.g. job title)
- `bio` ‚Äî short description text
- `badge` ‚Äî JSX element next to the name (e.g. `<Badge>Pro</Badge>`)
- `coverImage` ‚Äî banner image URL
- `stats` ‚Äî `ProfileStat[]` ‚Äî horizontal stat items
- `actions` ‚Äî action button(s) on the right side
- `infoItems` ‚Äî JSX with extra info below the subtitle
- `tabs` ‚Äî `ProfileTab[]` ‚Äî tabbed content panels
- `activeTab` ‚Äî controlled active tab ID
- `onTabChange` ‚Äî `(id: string) => void` ‚Äî callback when tab changes

### `ProfileStat`

- `label` ‚Äî stat label (e.g. "Followers")
- `value` ‚Äî `string` | `number` ‚Äî stat value

### `ProfileTab`

- `id` ‚Äî unique tab identifier
- `label` ‚Äî `JSX.Element` ‚Äî tab trigger content
- `body` ‚Äî `JSX.Element` ‚Äî tab panel content
