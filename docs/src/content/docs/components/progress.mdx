The progress component exposes a root meter, optional grouping wrapper, and label slots aligned with Kobalte’s progress primitive.

## Usage

```tsx
import {
  Progress,
  ProgressGroup,
  ProgressLabel,
  ProgressValueLabel,
} from "shadcn-solid-components/components/progress"
```

Use `Progress` for the track and fill; add `ProgressLabel` and `ProgressValueLabel` when you need visible status text.

## Exports

- `Progress`, `ProgressGroup`, `ProgressLabel`, `ProgressValueLabel`
- Types: `ProgressProps`, `ProgressGroupProps`, `ProgressLabelProps`, `ProgressValueLabelProps`
