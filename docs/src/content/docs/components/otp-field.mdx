Built on `@corvu/otp-field`, this component renders a segmented code entry UI with hidden or visible inputs per slot.

## Installation

```bash
pnpm add @corvu/otp-field
```

## Usage

```tsx
import {
  OTPField,
  OTPFieldGroup,
  OTPFieldInput,
  OTPFieldSeparator,
  OTPFieldSlot,
} from "shadcn-solid-components/components/otp-field"
```

Place one `OTPFieldSlot` per digit inside `OTPFieldGroup`, with a single `OTPFieldInput` for the actual value binding. Use `OTPFieldSeparator` between groups when needed.

## Exports

- `OTPField`, `OTPFieldGroup`, `OTPFieldSlot`, `OTPFieldSeparator`, `OTPFieldInput`
- Types: `OTPFieldProps`, `OTPFieldGroupProps`, `OTPFieldSlotProps`, `OTPFieldSeparatorProps`, `OTPFieldInputProps`
