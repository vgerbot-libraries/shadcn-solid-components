Tabs use KobalteŌĆÖs tabs primitive with orientation-aware layout, a sliding indicator, and theme hooks for list styling. Behavior (value, controlled mode, keyboard navigation) matches KobalteŌĆÖs API.

## Installation

Install `shadcn-solid-components` and satisfy peer dependencies (`solid-js`, Tailwind CSS v4, `tw-animate-css`).

## Usage

```tsx
import {
  Tabs,
  TabsContent,
  TabsIndicator,
  TabsList,
  TabsTrigger,
} from "shadcn-solid-components/components/tabs"
```

Place `TabsTrigger` elements inside `TabsList` (with `TabsIndicator` as needed), and pair each trigger with a `TabsContent` using the same `value`. See [Kobalte Tabs](https://kobalte.dev/docs/core/components/tabs) for props such as `value`, `onChange`, and `orientation`.

## Exports

- `Tabs`, `TabsList`, `TabsTrigger`, `TabsIndicator`, `TabsContent`
- Types: `TabsProps`, `TabsListProps`, `TabsTriggerProps`, `TabsIndicatorProps`, `TabsContentProps`
