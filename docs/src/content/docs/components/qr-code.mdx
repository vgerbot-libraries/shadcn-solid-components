`QrCode` renders a scannable SVG QR image from a string value, with customizable size, foreground/background colors, and error correction level.

## Usage

```tsx
import { QrCode } from "shadcn-solid-components/components/qr-code"
```

## Exports

- `QrCode`
- Types: `QrCodeProps`, `QrCodeErrorCorrectionLevel`

## Examples

### Custom colors and error correction

```tsx
<QrCode
  value="WIFI:T:WPA;S:Office-WiFi;P:supersecret123;H:false;;"
  size={192}
  fgColor="var(--primary)"
  bgColor="var(--muted)"
  errorCorrectionLevel="H"
/>
```
