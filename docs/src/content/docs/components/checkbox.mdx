import { Checkbox, CheckboxControl, CheckboxDescription, CheckboxInput, CheckboxLabel } from "shadcn-solid-components/components/checkbox"

Checkboxes follow Kobalte’s checkbox pattern: root, label, optional description, hidden input, control, and indicator parts.

## Installation

Install `shadcn-solid-components` and satisfy peer dependencies (`solid-js`, Tailwind CSS v4, `tw-animate-css`).

## Usage

```tsx
import {
  Checkbox,
  CheckboxControl,
  CheckboxDescription,
  CheckboxInput,
  CheckboxLabel,
} from "shadcn-solid-components/components/checkbox"
```
