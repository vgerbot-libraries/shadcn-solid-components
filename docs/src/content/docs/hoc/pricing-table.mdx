`PricingTable` renders a grid of plan cards with a monthly/yearly billing toggle, feature
checklists, a "Most Popular" badge, discount badges, and CTA buttons. The grid adapts
automatically to 1–4+ columns.

## Usage

```tsx
import { PricingTable } from "shadcn-solid-components/hoc/pricing-table"

<PricingTable
  plans={[
    {
      name: "Free",
      priceMonthly: 0,
      priceYearly: 0,
      cta: "Get started",
      features: [
        { text: "1 project", included: true },
        { text: "Custom domain", included: false },
      ],
    },
    {
      name: "Pro",
      priceMonthly: 19,
      priceYearly: 190,
      isPopular: true,
      cta: "Start free trial",
      features: [
        { text: "Unlimited projects", included: true },
        { text: "Custom domain", included: true },
      ],
    },
  ]}
/>
```

The billing toggle switches between `priceMonthly` and `priceYearly` on every plan simultaneously.
Plans marked with `isPopular` receive a prominent border and a "Most Popular" badge.

### Custom currency and default billing

```tsx
<PricingTable currency="€" defaultBilling="yearly" plans={plans} />
```

### Per-plan theme colour

Use `themeColor` on popular plans to tint the border, badge, and discount text:

```tsx
<PricingTable
  plans={[
    {
      name: "Pro",
      priceMonthly: 29,
      priceYearly: 290,
      isPopular: true,
      themeColor: "#6366f1",
      cta: "Upgrade",
      features: [{ text: "All features", included: true }],
    },
  ]}
/>
```

## API

### `PricingTableProps`

Extends `div` props.

- `plans` — `PricingPlan[]` — array of plan definitions
- `defaultBilling` — `'monthly'` | `'yearly'` (default `'monthly'`)
- `currency` — `string` — currency symbol (default `'$'`)
- `header` — `JSX.Element` — content above the billing toggle
- `footer` — `JSX.Element` — content below the plan grid
- `locale` — `Partial<PricingTableLocale>` — locale string overrides

### `PricingPlan`

- `name` — plan name (e.g. "Free", "Pro")
- `description` — optional short description
- `priceMonthly`, `priceYearly` — numeric price values
- `features` — `PricingFeature[]` — main feature list
- `additionalFeatures` — `PricingFeature[]` — secondary feature section
- `isPopular` — `boolean` — highlights with border and badge
- `themeColor` — `string` — hex colour for popular border/badge tinting
- `discount` — `string` — discount badge text (e.g. "17% off")
- `cta` — `string` — CTA button label
- `ctaVariant` — button variant override (default: `'default'` for popular, `'outline'` for others)
- `ctaClass` — additional CSS class(es) for the CTA button
- `ctaInfo` — `string` | `JSX.Element` — info text below the CTA (strings get an auto info icon)
- `billingText` — `string` — custom billing subtitle (always shown when set)
- `footerAction` — `JSX.Element` — custom content at bottom of card footer
- `onSelect` — `() => void` — CTA click callback

### `PricingFeature`

- `text` — feature description
- `included` — `boolean` — check or cross icon

## i18n

Locale key: `PricingTable`. Overridable strings:

| Key                    | Default              |
| ---------------------- | -------------------- |
| `monthly`              | Monthly              |
| `yearly`               | Yearly               |
| `popular`              | Most Popular         |
| `perMonth`             | mo                   |
| `perYear`              | yr                   |
| `billedYearly`         | `(currency, amount) => string` |
| `additionalFeatures`   | Additional Features  |
