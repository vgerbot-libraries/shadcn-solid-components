`DatePickerField` wraps the lower-level `DatePicker*` components from `shadcn-solid-components/components/date-picker` with form-style labeling, errors, optional presets sidebar, and calendar chrome (day/month/year views, optional today button and month/year selects).

## Installation

Install `shadcn-solid-components` and satisfy peer dependencies (`solid-js`, Tailwind CSS v4, `tw-animate-css`).

## Usage

```tsx
import { DatePickerField } from "shadcn-solid-components/hoc/date-picker"

<DatePickerField label="Birthday" selectionMode="single" />
```

Root props such as `value`, `defaultValue`, `onValueChange`, `selectionMode`, `inline`, and locale-related options are forwarded to the `DatePicker` primitive from `shadcn-solid-components/components/date-picker`. `DateValue` is re-exported for typing preset values.

`presets` can assign `value` from a `DateRangePreset` string such as `last7Days` for quick range
selection. When validation fails, pass `error` so the field switches to error styling and hides the
description text.

## API

### `DatePickerFieldProps`

Extends `div` props (minus `onChange`) plus a picked subset of Ark `DatePickerRootProps` (value, selection mode, views, locale, `inline`, etc.—see `src/hoc/date-picker/index.tsx` for the full picked list).

HOC-specific fields include:

- `label`, `description`, `error`, `required`, `placeholder`, `clearable`
- `presets` — `{ label, value }[]` where `value` is `DateValue[]` or a `DateRangePreset` string key
- `i18n` — partial locale for HOC strings
- `contentClass`, `showMonthYearSelect`, `showTodayButton`, `calendarHeader`, `calendarFooter`

### `DatePickerPreset`

- `label`, `value` — `DateValue[]` or `DateRangePreset`

### `DateRangePreset`

String union such as `'thisWeek'`, `'last7Days'`, `'thisMonth'`, etc. (see source for the full set).

## Exports

- `DatePickerField` — component
- `DatePickerFieldProps`, `DatePickerPreset`, `DateRangePreset` — types
- `DateValue` — re-export from Ark UI date picker
