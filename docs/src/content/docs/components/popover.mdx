Popovers show **non-modal** content (compact forms, filters, extra actions) in a positioned layer. They stay open until dismissed (click outside, Escape, or your own controls). Prefer **tooltips** for short hints on hover/focus, and **dialogs** when you need a modal focus trap and page-blocking overlay.

## Usage

```tsx
import {
  Popover,
  PopoverContent,
  PopoverPortal,
  PopoverTrigger,
} from "shadcn-solid-components/components/popover"
```

Wrap the anchor in `PopoverTrigger` and place the surfaced UI inside `PopoverContent` (typically under `PopoverPortal` when portaling is required by the primitive). Size and padding are usually controlled with utilities on `PopoverContent`.

Use `open` and `onOpenChange` on `Popover` when you need to control visibility from outside (for example closing after submit).

### Controlled open

Drive the panel with `open` / `onOpenChange` when another action should close it or when multiple UI pieces must stay in sync.

<ComponentPreview name="Popover Controlled">
  <div class="mb-4 w-full">
    <PopoverControlledDemo />
  </div>
  ```tsx
  import { createSignal } from "solid-js"

  import { Button } from "shadcn-solid-components/components/button"
  import {
    Popover,
    PopoverContent,
    PopoverPortal,
    PopoverTrigger,
  } from "shadcn-solid-components/components/popover"

  const PopoverControlledDemo = () => {
    const [open, setOpen] = createSignal(false)

    return (
      <Popover open={open()} onOpenChange={setOpen}>
        <PopoverTrigger<typeof Button>
          as={(props) => (
            <Button variant="outline" {...props}>
              {open() ? "Close" : "Open"} popover
            </Button>
          )}
        />
        <PopoverPortal>
          <PopoverContent class="w-72">
            <p class="text-muted-foreground mb-3 text-sm">
              Controlled example: the trigger label reflects open state.
            </p>
            <Button
              class="w-full"
              variant="secondary"
              onClick={() => setOpen(false)}
            >
              Close from inside
            </Button>
          </PopoverContent>
        </PopoverPortal>
      </Popover>
    )
  }

  export default PopoverControlledDemo
  ```
</ComponentPreview>
