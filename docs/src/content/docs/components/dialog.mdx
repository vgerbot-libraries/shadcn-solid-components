Dialogs use Kobalteâ€™s dialog primitive with portal, overlay, content, header/footer regions, title, description, trigger, and an explicit close button part.

## Usage

```tsx
import {
  Dialog,
  DialogCloseButton,
  DialogContent,
  DialogDescription,
  DialogFooter,
  DialogHeader,
  DialogPortal,
  DialogTitle,
  DialogTrigger,
} from "shadcn-solid-components/components/dialog"
```

## Exports

- `Dialog`, `DialogPortal`, `DialogTrigger`, `DialogCloseButton`, `DialogContent`, `DialogHeader`, `DialogFooter`, `DialogTitle`, `DialogDescription`
- Types: `DialogProps`, `DialogTriggerProps`, `DialogCloseButtonProps`, `DialogContentProps`, `DialogHeaderProps`, `DialogFooterProps`, `DialogTitleProps`, `DialogDescriptionProps`
