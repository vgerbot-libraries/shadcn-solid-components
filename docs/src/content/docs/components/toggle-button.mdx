`ToggleButton` wraps Kobalte’s toggle button with `cva` variants (`variant`, `size`) and theme-aware classes. It is the same visual language as `ToggleGroupItem`, which consumes matching variant props from context.

## Installation

Install `shadcn-solid-components` and satisfy peer dependencies (`solid-js`, Tailwind CSS v4, `tw-animate-css`).

## Usage

```tsx
import { ToggleButton, toggleButtonVariants } from "shadcn-solid-components/components/toggle-button"
```

Use `pressed`, `onChange`, and other Kobalte toggle-button props for state. Optional `variant` and `size` match those used by `ToggleGroup`.

## Exports

- `ToggleButton`, `toggleButtonVariants`
- Types: `ToggleButtonProps`
