import { Button } from "shadcn-solid-components/components/button"

The button component supports multiple visual variants and sizes through `buttonVariants`. It can render as different underlying elements via the polymorphic `as` prop.

## Usage

```tsx
import { Button, buttonVariants } from "shadcn-solid-components/components/button"
```

Use `Button` for interactive actions. Import `buttonVariants` when you need the same styles on a non-button element (for example an anchor styled as a button).
