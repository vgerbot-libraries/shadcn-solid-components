The accordion wraps Kobalte’s accordion primitive with slots and styling for triggers, headers, and animated content regions.

## Usage

```tsx
import {
  Accordion,
  AccordionContent,
  AccordionItem,
  AccordionTrigger,
} from "shadcn-solid-components/components/accordion"
```

Compose one or more `AccordionItem` children, each with an `AccordionTrigger` and `AccordionContent`. Behavior and accessibility follow Kobalte’s accordion API.

## Exports

- `Accordion`, `AccordionItem`, `AccordionTrigger`, `AccordionContent`
- Types: `AccordionProps`, `AccordionItemProps`, `AccordionTriggerProps`, `AccordionContentProps`

        </AccordionContent>
      </AccordionItem>
      <AccordionItem value="item-3">

        <AccordionContent>
          Yes. It's animated by default, but you can disable it if you prefer.
        </AccordionContent>
      </AccordionItem>
    </Accordion>
  )
}

export default AccordionDemo
