Drawers are built on `@corvu/drawer` with styled trigger, close control, content shell, header, footer, label, and description slots.

## Usage

```tsx
import {
  Drawer,
  DrawerClose,
  DrawerContent,
  DrawerDescription,
  DrawerFooter,
  DrawerHeader,
  DrawerLabel,
  DrawerPortal,
  DrawerTrigger,
} from "shadcn-solid-components/components/drawer"
```

## Exports

- `Drawer`, `DrawerPortal`, `DrawerTrigger`, `DrawerClose`, `DrawerContent`, `DrawerLabel`, `DrawerDescription`, `DrawerHeader`, `DrawerFooter`
- Types: `DrawerProps`, `DrawerTriggerProps`, `DrawerCloseProps`, `DrawerContentProps`, `DrawerLabelProps`, `DrawerDescriptionProps`, `DrawerHeaderProps`, `DrawerFooterProps`
