The table module wraps semantic `<table>` markup in a horizontal scroll container and applies consistent typography, borders, and hover states. Use it for static or manually rendered grids; for large data sets with sorting and filtering, see **TanStack table** or **Tabulator table** components.

## Installation

Install `shadcn-solid-components` and satisfy peer dependencies (`solid-js`, Tailwind CSS v4, `tw-animate-css`).

## Usage

```tsx
import {
  Table,
  TableBody,
  TableCaption,
  TableCell,
  TableFooter,
  TableHead,
  TableHeader,
  TableRow,
} from "shadcn-solid-components/components/table"
```

Compose `TableHeader` / `TableBody` / `TableFooter` with `TableRow`, `TableHead`, and `TableCell` as in standard HTML. The root `Table` wraps the `<table>` in a `div` with `overflow-x-auto`.

## Exports

- `Table`, `TableHeader`, `TableBody`, `TableFooter`, `TableRow`, `TableHead`, `TableCell`, `TableCaption`
- Types: `TableProps`, `TableHeaderProps`, `TableBodyProps`, `TableFooterProps`, `TableRowProps`, `TableHeadProps`, `TableCellProps`, `TableCaptionProps`

              <TableCell>{invoice.paymentMethod}</TableCell>
              <TableCell class="text-right">{invoice.totalAmount}</TableCell>

      <TableFooter>
        <TableRow>
          <TableCell colSpan={3}>Total</TableCell>
          <TableCell class="text-right">$2,500.00</TableCell>
        </TableRow>
      </TableFooter>
    </Table>
  )
}

export default TableDemo
